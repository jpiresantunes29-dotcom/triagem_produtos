# Triagem de Produtos

Projeto em Python para cadastrar e classificar pedidos com:

- validacao de entradas
- menu interativo
- regras de classificacao
- relatorios simples

## Estrutura

- `src/main.py`
- `src/validacoes.py`
- `src/regras.py`
- `src/relatorios.py`

## Como executar

Abra a pasta do projeto no VS Code e rode:

```bash
python src/main.py
```

Se necessario, tente:

```bash
py src/main.py
```
